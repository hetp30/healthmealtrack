# KissanVarg - Organic Farming Website

A beautiful, responsive website showcasing traditional organic farming practices with modern web design and smooth animations.

## 🌱 Features

### Design & Theme
- **Traditional Organic Farming Theme**: Earth-toned color palette with natural greens, browns, and gold accents
- **Responsive Design**: Fully responsive across all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, intuitive interface with excellent user experience
- **Accessibility**: Semantic HTML structure and keyboard navigation support

### Animations & Interactions
- **Smooth Scroll Animations**: Elements fade in as you scroll
- **Floating Elements**: Animated icons in the hero section
- **Counter Animations**: Animated statistics counters
- **Hover Effects**: Interactive hover animations on cards and buttons
- **Parallax Effects**: Subtle parallax scrolling for depth
- **Modal Popups**: Interactive gallery modals with smooth transitions

### Sections
1. **Hero Section**: Eye-catching landing with animated elements
2. **About Section**: Company story with animated statistics
3. **Products Section**: Showcase of organic products with hover effects
4. **Methods Section**: Timeline of traditional farming methods
5. **Gallery Section**: Interactive photo gallery with modals
6. **Contact Section**: Contact form with validation
7. **Footer**: Newsletter signup and social links

### Functionality
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Form Validation**: Contact and newsletter forms with validation
- **Smooth Scrolling**: Smooth navigation between sections
- **Notification System**: Toast notifications for form submissions
- **Gallery Modals**: Click to view detailed gallery items
- **Active Navigation**: Highlights current section in navigation

## 🎨 Color Palette

The website uses a carefully chosen traditional organic farming color palette:

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. The website is ready to use!

### File Structure
```
KissanVarg/

```

## 📝 Customization

### Colors
Modify the CSS custom properties in `:root` to change the color scheme:

```css
:root {
    --primary-green: #2d5a27;
    --secondary-green: #4a7c59;
    --accent-gold: #d4af37;
    /* ... other colors */
}
```

### Content
- Update text content in `index.html`
- Modify statistics in the About section
- Add/remove gallery items
- Update contact information

### Animations
- Adjust animation durations in CSS
- Modify keyframe animations
- Change scroll trigger thresholds

## 🔧 Development

### Adding New Sections
1. Add HTML structure to `index.html`
2. Add corresponding CSS styles to `style.css`
3. Add any JavaScript functionality to `script.js`
4. Update navigation if needed

### Adding New Animations
1. Define CSS keyframes
2. Add animation classes
3. Use Intersection Observer for scroll-triggered animations
